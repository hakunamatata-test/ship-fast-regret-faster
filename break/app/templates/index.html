MCP Attack-Pattern Workshop

Demos run as MCPJam challenges. Add each server in MCPJam (STDIO) and chat with the assistant to find the flag. See break/CHALLENGES.md for setup and commands.

  • 01 — Oops, That Was Private Secret exposure (flight booking).
  • 02 — Deputy in the Middle Man-in-the-middle / proxy.
  • 03 — Looks Legit to Me Tool poisoning via metadata/description.
  • 04 — Name Your Poison Tool naming / impersonation.
  • 05 — Behind the Curtain Prompt injection via tool parameters.
  • 06 — Forward Thinking Tool chaining / exfiltration.

Run each challenge with: uv run --directory <path>/break <challenge-folder>/challenge.py. Replace <path> with your repo path and <challenge-folder> with e.g. 04-name-your-poison.
